# Mantle in Swift
Example use Mantle in Swift project. 

## Mantle Use Example
Mantle use example in:
- MSModelManager.swift
- MSNews.swift

## Getting started
1. Install Carthage 
https://github.com/Carthage/Carthage
2. Install frameworks: Mantle, Alamofire, SwiftyJSON
```
$ cd project_dir
$ carthage update --platform iOS
```
